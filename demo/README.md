# Accelerometry Annotation Tool — Live Demo

Log in with **demo_admin / demo** or **demo_user / demo**.

Data is derived from real accelerometer recordings (UCI Accelerometer & Gyro
Mobile Phone Dataset, CC BY 4.0), resampled and composed into short demo files.
See the [main repository](https://github.com/TavoloPerUno/py_visualize_accelerometry)
for details.
